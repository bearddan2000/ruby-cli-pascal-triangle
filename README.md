# ruby-cli-pascal-triangle

## Description
Calculates binomial coefficient for a
given number, limit 10.

## Tech stack
- ruby 2.4

## Docker stack
- alpine:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
